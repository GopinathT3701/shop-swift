# 🛒 Full Stack E-Commerce Platform

A complete **Full Stack E-Commerce Web Application** that allows users to browse products, manage their shopping cart, place orders, and securely authenticate.  
The platform also includes product management features for handling inventory and orders efficiently.

This project demonstrates modern **full stack web development practices**, combining a responsive frontend with a scalable backend and database integration.

---

## 🚀 Features

### User Features
- User registration and login
- Secure authentication
- Browse products
- Add products to cart
- Update or remove cart items
- Place orders

### Admin Features
- Add new products
- Update product details
- Delete products
- Manage inventory
- View orders

---

## 🛠 Tech Stack

### Frontend
- React
- Tailwind CSS
- Axios
- React Router

### Backend
- Node.js
- Express.js

### Database
- MySQL / MongoDB

### Other Tools
- JWT Authentication
- REST API
